1. Configuration du projet :
   dans le dossier : git init
   git config -l 
   changer l'identité : git config --global user.name "Matis Chabanat"

2. Mise en place
   touch index.html 
   code .	// lancer VSCode le dossier courant

3. Modification et mise a jour du projet
   git add index.html
   git commit -m "Création squelette fichier index.html"

4. Lister les commit
   git log

5. Utiliser une branche : 
   git branch nomBranche 	// Tirer une branche
   git branch -d nomBranche	//Suppr la branche si elle a été fusionnée
   git branch -D nomBranche	//Suppr la branche si elle a pas été fusionnée
   git branch 	//Lister les branches
   git switch nomBranche 	//Changer de branche
   git switch - 	//revenir a la branche precedente
   git switch -c nomBranche 	//Switcher sur une nouvelle branche

6. Rappatrier du code sur master
   git switch master
   git diff master..nomBrancheAFusionner //Jeter un oeil aux differences de code
   git merge nomBrancheAFusionner
